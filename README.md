# نظام التنبؤ المبكر بالتعثر الأكاديمي

## 📋 وصف المشروع
نظام ذكي لتحليل بيانات الطلاب من Blackboard والتنبؤ المبكر بالتعثر الأكاديمي.

## 🗂️ هيكل المشروع
```
├── index.html              ← نقطة الدخول الرئيسية
├── start.bat               ← سكريبت التشغيل (Windows)
├── frontend/
│   ├── index.html          ← واجهة المستخدم
│   ├── style.css           ← التصميم
│   └── app.js              ← منطق الواجهة
└── backend/
    ├── app.py              ← خادم Flask
    ├── analyzer.py         ← محرك التحليل والتنبؤ
    └── requirements.txt    ← المتطلبات
```

## 🚀 طريقة التشغيل

### المتطلبات
- Python 3.8+
- pip

### التشغيل
```bash
# 1. تثبيت المتطلبات
cd backend
pip install -r requirements.txt

# 2. إعداد الإيميل (اختياري)
# قم بتحرير ملف .env في مجلد backend وأدخل كلمة مرور بريدك
# لـ Gmail، استخدم كلمة مرور التطبيق من: https://myaccount.google.com/apppasswords

# 3. تشغيل الخادم
python app.py
```

### 3. فتح الواجهة
افتح المتصفح على: `http://localhost:5000`  
أو افتح مباشرة: `frontend/index.html`

## 📊 الملفات المطلوبة

### Gradebook.xlsx
- اسم الطالب
- الدرجة الكلية
- درجات الاختبارات (أعمدة متعددة)

### Analytics.xlsx
- اسم الطالب
- التقدير الكلي
- تواريخ الاستحقاق الفائتة
- عدد الساعات المستغرقة
- عدد الأيام منذ آخر وصول

## 🎯 مخرجات النظام
- **تقرير شامل** لكل طالب
- **مستوى الخطر**: حرج / مرتفع / متوسط / منخفض
- **درجة الخطر** (0-100)
- **مستوى التفاعل**: ممتاز / جيد / متوسط / ضعيف
- **اتجاه الأداء**: تحسن / مستقر / تراجع / متذبذب
- **توصيات ذكية** مخصصة لكل طالب
- **تنزيل تقرير Excel** شامل بثلاث صفحات
